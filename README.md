# LAWGIVER MkII - Eaglercraft Resource Pack

Screen-accurate Judge Dredd Lawgiver pistol for Eaglercraft based on DREDD (2012).

## Features
-  7 Ammo Modes (FMJ, AP, INC, HE, HS, ST, RAPID)
-  Screen-Accurate LCD HUD
-  7 Unique Sound Effects
-  Reload & Mode Change Sounds

## Installation

1. Download `LAWGIVER_MkII.zip`
2. Place in your Eaglercraft resourcepacks folder:
   - **Windows**: `%appdata%\.eaglercraft\resourcepacks\`
   - **macOS**: `~/Library/Application Support/.eaglercraft/resourcepacks/`
   - **Linux**: `~/.eaglercraft/resourcepacks/`
3. Activate in Resource Packs menu

## Usage (with EaglerForge mod)
- **R** - Cycle ammo mode
- **F** - Change fire mode
- **H** - Toggle HUD
- **Mouse Click** - Fire
